# Cloud Native On-Prem Java Operator - KDC layout

Slides

Makes with [Slidev](https://slidev.org/) and theme [NeverSink](https://github.com/gureckis/slidev-theme-neversink)

```bash
# install dependencies
yarn install
# start dev server
yarn dev
# build slides for production
yarn build
# export slides to pdf
yarn export
```

It's available in [GitHub Pages](https://jonesbusy.github.io/kcd-cloud-native-on-prem/)
